# To-Do List Full Stack App

This is a full-stack To-Do List application made using React, Node.js, Express.js, and MongoDB.

## Features

- Add a new task
- View all tasks
- Update task details
- Mark task as complete or pending
- Delete a task
- Search tasks
- Frontend and backend connected using Axios

## Tech Stack

Frontend:
- React
- Axios
- CSS

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
